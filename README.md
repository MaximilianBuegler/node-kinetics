# node-kinetics
Node.js package for transforming accelerometer and gyroscope data

# Notes
    Assumes all input data to be 2D arrays
    
    [[x1, y1, z1], [x2, y2, z2],...]
    [[pitch1, roll1, yaw1]. [pitch2, roll2, yaw2],... ] (in Radians)
